Welcome to the Concepts Worksheet

For each of 208 canonical concepts, make one decision per entry: keep, merge, new, or drop. Most entries have a clear Claude-recommended verdict — just confirm it. A handful need your judgment.

How it works:

  • Your progress saves automatically in this browser. The bottom-left badge shows the status.
  • Concepts are organized by frequency (§11 high-impact → §12 mid-tier → §13 long-tail). High-impact deserves the most attention; long-tail can be skimmed.
  • Five Cluster signal cards on the right ask cross-entry questions (writing cluster, reading cluster, measurement cluster, figurative-language cluster, Indigenous cross-field). Answer those alongside the per-entry verdicts.
  • When you're done (or want a backup), click Save & Export ↓. It downloads a markdown file you send back to the project maintainer.
  • Three entries flag theme overlap with the themes worksheet — your verdict here is still useful; the themes-side adjudication happens later.

Local saving is tied to this browser on this computer. To move work between devices, use Save & Export and Import from the ⚙ Advanced menu.
