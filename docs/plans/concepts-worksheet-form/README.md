# Concepts Worksheet — How to Use

This folder holds an interactive tool for reviewing our lesson concepts. The whole tool is a single file — everything it needs is already inside it. No install, no internet connection, nothing to set up.

## Opening it

Double-click **`concepts-worksheet-tool.html`**. It opens in your web browser (Chrome, Safari, Firefox, and Edge all work). If it doesn't open on its own, right-click the file and choose **Open With** → your browser.

The first open takes a second or two while the page loads; after that it's instant.

## What you're doing

You're going through **208 concepts** — the words we use to tag and find lessons, things like *plant parts*, *measurement*, or *figurative language* — one at a time. For each one you decide what should happen to it:

- **Keep it** — it's a good concept; leave it as is
- **Fold it into another** — it overlaps with a similar concept; combine the two
- **Add a new one** — something's missing from the list and should be added (rare)
- **Remove it** — it isn't useful for tagging; drop it

Those decisions give the team a clean, trusted list to tag lessons against.

## Three kinds of decisions

The tool sorts every concept into the right kind for you and shows a small label on each one. The counts along the top tell you how many of each are left.

- **Quick check** — Most concepts come with a suggested answer. Glance at it, then either agree or pick something different. Each suggestion comes with a short reason, and a **more** link if you want the full explanation.
- **Your call** — A handful come with no suggestion. These are the ones worth a careful look — you decide from a blank slate.
- **Group decision** — A few small groups of closely-related concepts are decided together. You make one choice for the group and it carries down to each concept in it. (If you later make a different choice for one of those concepts on its own, the tool flags it — both on the concept itself and on the **Review so far** screen — so you can double-check.)

## Moving through it

**Save · Next →** moves you forward; **← Previous** steps back. You don't have to finish in order, and you don't have to do it all in one sitting. There's also a search box at the top — start typing a concept's name to jump straight to it.

If a concept stumps you, click **Decide later** to set it aside and keep going. The **Review so far** screen lists everything you've set aside so you can return to it.

### Keyboard shortcuts (optional)

If you'd rather not reach for the mouse:

- **Enter** — confirm and move to the next one (on *Your call* and *Group decision* steps, make your choice first)
- **←** — go back
- **L** — decide later
- **K** keep it · **M** fold it into another · **N** add a new one · **D** remove it

(Shortcuts pause while you're typing in a note or text box.)

## Review so far

Click **Review so far** at the top any time to see everything you've decided, grouped by outcome — kept, folded into another, added, removed, and your group decisions. Two groups are worth a glance:

- **Decide later** — the concepts you set aside; click any to jump back to it
- **Differs from a group decision** — places where your choice for a single concept doesn't match the group decision it belongs to. Usually that's on purpose, but it's worth a quick look.

You can export your work straight from this screen, too.

## How your work is saved

Your answers save automatically as you go, right inside your browser on this computer — the **Saved locally** badge at the top of the page confirms it. You can close the tab and come back later; your work will be waiting.

A few things to know about saving in the browser:

- Your progress lives in **this browser on this computer**. Another browser, or another machine, won't have it automatically.
- Clearing your browser history (with site data) will erase it.
- Private / incognito windows often discard everything when you close them — either avoid private mode for this, or export often (below).

## Sending your work to the team

Open the **⚙ Advanced** menu in the top-right corner and choose **Save & Export markdown ↓**. A file downloads to your usual Downloads folder (it keeps the worksheet's own name) — that's the file to send back to the team. It's the full worksheet with your choices filled in.

(If you export before making any decisions, the file comes out identical to the original — nothing changes until you actually decide something.)

To pick up where a saved file left off — for example on another computer — open the tool and choose **Import a saved markdown file…** from the same ⚙ Advanced menu.

## Disagreeing with a suggestion

The suggested answers are starting points, not final decisions — pushback is the whole point. When you pick something different from the suggestion, a small note box opens so you can jot down why. Those notes travel with your export, so the team sees your reasoning.

## If something seems off

- Your saved answers are small, but in private / incognito mode some browsers throw them away when the window closes — so export often if you work that way.
- Some workplace computers block file uploads; if **Import** won't open a file, try it from a personal browser profile.
